1. window.onload 和 document.onload 的区别
window.onload 在结构样式加载完触发
document.onload 在结构样式加载完并执行完图片样式文件才触发

2. HTTPS 校验过程
校验过程：

1. 浏览器请求证书
2. 服务端返回 HTTPS 证书
3. 客户端验证证书是否合法

传输阶段：
1. 客户端生成随机数
2. 通过公钥加密随机数，并传给服务端
3. 服务端通过私钥对随机数解密，并将随机数加密返回数据

10. 函数节流
```
function throttle (fn, delay) {
    var previous = 0
    var _this = this
    var args = arguments
    var now = new Date()
    if (now - previous > delay) {
        fn.apply(_this, args)
        now = previous
    }
}
```
