4. http、http 缓存
+ 强制缓存
    + 强制缓存，在缓存数据未失效的情况下，可以直接使用缓存数据，不需要再请求服务器
    + Cache-Control
        + max-age
        + public
        + private
        + no-cache
+ 对比缓存
    + 浏览器第一次请求数据时，服务器会将缓存标识与数据一起返回给浏览器，浏览器将二者备份至缓存数据库中。
    + 当浏览器再次请求数据时，浏览器将备份的缓存标识发送给服务器，服务器根据缓存标识进行判断，判断成功后，返回304状态码，通知客户端比较成功，可以使用缓存数据。
